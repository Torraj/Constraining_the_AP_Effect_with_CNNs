# Constraining_the_AP_Effect_with_CNNs
Contains codes relevant for using convolutional neural netorks (CNNs) to constrain a proxy for the Alcock-Paczynski effect in simulated galaxy catalogs. Codes included can generate mock galaxy catalogs (through multiple methods), artificially stretch catalogs consistent with the AP effect, and construct/train a CNN to identify that stretch 

The folder "Gaussian_Random_Field_Mocks" contains any work using mock galaxy catalogs generated from Gaussian random fields (GRFs) to train CNNs. "Norm_Gaussian_Field_test.ipynb" is a notebook that generates GRFs from a realistic matter power spectrum that has been stretched by some factor. The notebook also contains all the code necessary to construct and train a CNN from the generated GRFs. The folder "Galaxy_Field_Modules_Test" contains a similar notebook for training a CNN from mock galaxy catalogs generated from these stretched GRFs. The only difference is that in this notebook, I experiment with importing functions from .py files rather than defining all my functions within the notebook itself. I do this because many of the notebooks in this repository contain the same shared code, so in hindsight, it would have been more efficient to define that shared code in .py files and then import them into the many notebooks of this repository. The "Verification" notebooks are used to visualize the effectiveness of the trained CNNS in recognizing the factor by which fields have been stretched. Specifically, they plot the actual stretch of a field against the CNN's estimated stretch for 1000 different fields, and those plots are useful for seeing biases in trained CNNs.

The folder "HOD_Galaxy_Catalogs_From_Quijote_Halos" contains any work using galaxy catalogs generated using Halo Occupation Distribution (HOD) codes applied to simulated dark matter halos from the Quijote catalog. https://quijote-simulations.readthedocs.io/en/latest/sancho.html
